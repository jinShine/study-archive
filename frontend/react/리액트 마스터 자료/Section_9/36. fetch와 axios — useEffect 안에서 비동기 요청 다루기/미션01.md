# 🟩 useEffect로 HTTP 요청 보내기 (JSONPlaceholder 실습)

## 📝 설명

이제 실제로 React에서 HTTP 요청을 보내 데이터를 불러오는 연습을 해봅니다.
이번 미션에서는 JSONPlaceholder API를 사용해 게시글 목록을 불러오고 화면에 표시합니다.
이 API는 무료로 사용할 수 있으며, 네트워크 통신의 전체 구조를 연습하기에 완벽한 환경입니다.

요청 주소는 다음과 같습니다.

`https://jsonplaceholder.typicode.com/posts`

## 🎯 미션 목표

- React의 `useEffect` 안에서 HTTP 요청을 올바른 시점에 실행할 수 있다.
- 요청이 완료되면 `setState`로 데이터를 화면에 반영한다.
- 렌더링 단계와 데이터 요청 단계를 분리해, 화면이 멈추지 않도록 한다.

## 🧱 Base Code

```jsx
import { useState, useEffect } from "react";

export default function PostList() {
  const [posts, setPosts] = useState([]);

  // TODO: 컴포넌트가 처음 나타날 때 한 번만 게시글 목록 불러오기

  return (
    <div>
      <h2>📄 게시글 목록</h2>
      {posts.length === 0 ? (
        <p>로딩 중...</p>
      ) : (
        <ul>
          {posts.map((post) => (
            <li key={post.id}>
              <strong>{post.title}</strong>
              <p>{post.body}</p>
            </li>
          ))}
        </ul>
      )}
    </div>
  );
}
```

## 📌 미션의 의의

이 미션을 통해 렌더링 → `useEffect` 실행 → `fetch` 요청 → 응답 수신 → state 업데이트 → 재렌더링의 순서를 직접 확인합니다.
`useEffect`는 DOM이 모두 반영된 "렌더링 직후"에 실행되므로, 화면이 멈추지 않고 데이터가 도착하면 자연스럽게 갱신됩니다.

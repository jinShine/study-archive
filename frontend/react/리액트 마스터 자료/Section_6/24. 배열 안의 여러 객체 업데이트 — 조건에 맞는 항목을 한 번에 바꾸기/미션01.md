# 🟪 쿠폰 한 번에 적용! — 조건에 맞는 여러 항목을 불변성으로 일괄 업데이트하기

## 📝 설명

배열 state에서 여러 객체를 한 번에 바꾸기를 연습합니다. 원본을 직접 수정하지 말고, `map()`으로 새 배열을 만들어 교체해야 React가 변화를 감지합니다.

## ✅ 요구사항

- 장바구니 `items` 상태는 아래 구조를 사용합니다.

```jsx
[
  { id: 1, name: "노트북",  price: 1200000, category: "electronics" },
  { id: 2, name: "키보드",  price:  50000,  category: "electronics" },
  { id: 3, name: "에코백",  price:  19000,  category: "fashion" }
]
```

- **전체 10% 할인 적용** 버튼을 누르면 모든 `price`를 `Math.floor(price * 0.9)`로 바꿉니다.
- **전자제품만 +5% 추가할인** 버튼을 누르면 `category === "electronics"` 인 항목만 `Math.floor(price * 0.95)`로 다시 갱신합니다.
- **2만원 미만 무료증정 태그 달기** 버튼을 누르면 `price < 20000`인 항목들에 `gift: true` 필드를 추가(없으면 생성)합니다.
- (옵션) **초기화** 버튼으로 초기 상태로 복원하세요.

## 🖥️ 출력 결과 예시

```text
# 초기
노트북 — 1200000원 (electronics)
키보드 — 50000원 (electronics)
에코백 — 19000원 (fashion)

# [전체 10% 할인] 클릭
노트북 — 1080000원
키보드 — 45000원
에코백 — 17100원

# [전자제품만 +5% 추가할인] 클릭
노트북 — 1026000원
키보드 — 42750원
에코백 — 17100원  # 그대로

# [2만원 미만 무료증정] 클릭
에코백 — 17100원 (🎁 gift)
```

## 💡 구현에 필요한 개념

- **불변성 유지(배열/객체)**: 원본 배열을 직접 바꾸는 `push`/`splice` 대신 `map()`으로 새 배열, 객체는 전개 연산자 `...` 로 얕은 복사 후 필요한 필드만 덮어쓰기.

- **조건부 일괄 수정**:

```jsx
setItems(items.map(it =>
  조건 ? { ...it, price: 새값 } : it
));
```

- **부분 필드 추가/갱신**: 존재하지 않는 필드도 `{ ...it, gift: true }`처럼 안전하게 추가.

- **참조 비교**: React는 "내용"이 아니라 참조 변경으로 업데이트를 감지합니다(새 배열/새 객체 생성 필수).

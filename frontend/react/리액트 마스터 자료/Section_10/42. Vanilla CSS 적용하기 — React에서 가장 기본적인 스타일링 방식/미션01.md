# ⬜ Vanilla CSS로 카드 스타일 전역 적용하기 — 전역 범위의 한계 체험하기

## 📝 설명

이번 미션에서는 React에서 CSS 파일을 `import`했을 때, 해당 스타일이 전체 프로젝트 전역에 영향을 미치는 원리를 직접 체험해 보시겠습니다.

여러분은 두 개의 상품 카드 컴포넌트를 만들고, `App.css`에 정의한 전역 규칙이 모든 컴포넌트에 동시에 적용되는 현상을 눈으로 확인하게 됩니다.
이를 통해 Vanilla CSS의 장점과 한계, 그리고 왜 이후 강의에서 CSS Modules이나 CSS-in-JS 같은 방식이 등장했는지를 명확히 이해하실 수 있습니다.

## 🖥️ 출력 결과 예시

```text
상품 목록

[ 사과 🍎 ]
가격: 2,000원
구매하기

[ 바나나 🍌 ]
가격: 3,000원
구매하기
```

두 카드 모두 같은 색상, 같은 테두리, 같은 버튼 스타일을 공유하게 됩니다.
특정 카드의 색만 다르게 하고 싶더라도, Vanilla CSS에서는 그게 쉽지 않다는 점을 직접 경험하실 수 있습니다.

## 💡 구현에 필요한 개념

- **전역 선택자(Global Selector)**: `div`, `button`, `.card` 등의 선택자는 프로젝트 전역에 영향을 미칩니다.
- **import 방식**: React에서는 HTML처럼 `<link>`로 불러오지 않고, `import "./App.css"` 형태로 모듈처럼 불러옵니다.
- **번들러(Webpack/Vite)**: CSS를 자바스크립트 모듈로 처리하고, 빌드 시 HTML `<style>` 태그 안에 자동으로 삽입합니다.
- **전역 충돌 문제**: 클래스 이름이 같거나 단순 태그 선택자를 사용하면, 여러 컴포넌트가 한꺼번에 영향을 받게 됩니다.
- **DevTools 활용**: "Styles" 패널에서 적용된 CSS의 출처와 우선순위를 꼭 확인하시면 좋습니다.

## 🧱 Base Code

### `src/App.css`

```css
/* App.css */
body {
  font-family: "Pretendard", sans-serif;
  background-color: #f9fafb;
  text-align: center;
  padding: 40px;
}

/* 전역 카드 스타일 */
.card {
  border: 2px solid #333;
  background-color: white;
  padding: 20px;
  width: 200px;
  margin: 10px auto;
  border-radius: 8px;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
}

/* 전역 버튼 스타일 */
button {
  background-color: seagreen;
  color: white;
  border: none;
  padding: 8px 12px;
  border-radius: 5px;
  cursor: pointer;
}

button:hover {
  background-color: darkgreen;
}
```

### `src/ProductCard.js`

```jsx
// ProductCard.js
export default function ProductCard({ name, price, emoji }) {
  return (
    <div className="card">
      <h2>{name} {emoji}</h2>
      <p>가격: {price.toLocaleString()}원</p>
      <button>구매하기</button>
    </div>
  );
}
```

### `src/App.js`

```jsx
// App.js
import "./App.css";
import ProductCard from "./ProductCard";

export default function App() {
  return (
    <div>
      <h1>상품 목록</h1>
      <ProductCard name="사과" price={2000} emoji="🍎" />
      <ProductCard name="바나나" price={3000} emoji="🍌" />
    </div>
  );
}
```

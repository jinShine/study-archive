# 🟫 내 취향 설정 폼 — Select·Checkbox·Radio를 모두 제어 컴포넌트로 만들기

## 📝 설명

하나의 설정 폼에서 `select`(단일 선택), `checkbox`(불리언), `radio`(그룹 단일 선택) 를 모두 제어 컴포넌트로 구현하세요. 입력값은 전부 React state에서 관리하며, 제출 시 현재 설정을 요약해서 보여줍니다.

## ✅ 요구사항

- 과일 선택: `select` (apple/banana/cherry)
- 뉴스레터 구독: `checkbox` (구독 여부)
- 알림 채널: `radio` (email/sms/push — 동일 `name`으로 그룹핑)
- "제출" 클릭 시 기본 동작을 막고(`preventDefault`), 아래에 요약 문장을 렌더링
- 모든 입력은 state ← `onChange` ← input 흐름을 갖는 제어 컴포넌트여야 함

## 🖥️ 출력 결과 예시

```text
[선택된 과일: banana]
[구독 여부: 구독 중]
[알림 채널: sms]
--------------------
제출 결과: 과일 banana / 구독 O / 채널 sms
```

## 💡 구현에 필요한 개념

- **제어 컴포넌트**: `value`(`select`/`radio`의 비교값), `checked`(`checkbox`) 를 state와 동기화

- **`onChange` 처리**:
  - `select`: `e.target.value`
  - `checkbox`: `e.target.checked`
  - `radio`: `e.target.value` + 같은 `name` 으로 그룹

- **불변성 & 리렌더링**: 매 업데이트 시 새 값/새 객체를 전달 → React가 참조 변경을 감지

- **폼 제출 제어**: `e.preventDefault()`로 새로고침 방지 후, state 기반 요약 출력

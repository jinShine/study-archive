# 🟫 회원가입 폼 완성하기 — 제출 흐름과 유효성 검사 구현

## 📝 설명

React에서 Form을 완성하려면 입력을 제어 컴포넌트로 관리하고, 제출 시 유효성 검사를 거쳐야 합니다.
이번 미션에서는 이름과 이메일, 약관 동의를 받는 회원가입 폼을 구현하며 `onSubmit`, `preventDefault`, 조건부 렌더링, `useState` 기반 검증 로직을 모두 직접 구성해 봅니다.

## ✅ 요구사항

- 이름(`input text`), 이메일(`input email`), 약관 동의(`checkbox`)를 각각 state로 관리
- 버튼 클릭 시 `onSubmit` 이벤트로 제출 처리
- 제출 초입에 `e.preventDefault()`로 새로고침 차단
- 이름이 비어있거나(`trim` 처리 포함), 이메일에 `@`가 없으면 제출 중단
- 위 조건에 걸리면 빨간색 에러 메시지를 표시 (error state로 조건부 렌더링)
- 모든 조건을 통과하면 `alert` 창으로 "가입 완료" 메시지 표시

## 🖥️ 출력 결과 예시

```text
이름 입력: [          ]
이메일 입력: [          ]
☐ 약관에 동의합니다
------------------
[제출 버튼 클릭 시]

❌ 사용자 이름을 입력하세요.
또는
❌ 올바른 이메일을 입력하세요.
------------------
✅ 제출 성공 시 → "홍길동님, 가입이 완료되었습니다!"
```

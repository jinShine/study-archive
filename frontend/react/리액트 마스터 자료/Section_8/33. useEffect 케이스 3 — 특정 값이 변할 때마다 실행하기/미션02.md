# 🟦 다크 모드 토글: useEffect(..., [dark])로 body 클래스 동기화

> [Section9]를 수강한 후 미션을 진행해주세요!

## 📝 설명

화면 상단의 스위치를 켜면 다크 모드, 끄면 라이트 모드로 전환합니다.
React 상태 `dark`가 바뀔 때마다 `document.body`에 `dark` 클래스를 추가/제거하여 전역 배경색을 바꾸세요.
컴포넌트가 사라질 때는 cleanup으로 `dark` 클래스를 제거해 깨끗한 종료를 보장합니다.

## ✅ 요구사항

- `dark`를 `useState(false)`로 관리한다.
- `useEffect`의 의존성 배열에 `[dark]`를 넣는다.
- `dark === true`면 `document.body.classList.add('dark')`, 아니면 `remove('dark')`.
- cleanup에서 항상 `document.body.classList.remove('dark')` 해 준다(컴포넌트 언마운트 대비).
- 화면에 라벨/토글 버튼과 현재 모드 표시를 렌더링한다.
- 간단한 전역 스타일(배경/글자색)을 함께 제공한다.

## 🧱 Base Code

```jsx
import { useEffect, useState } from "react";

// 전역 스타일(예시): 프로젝트 전역 CSS에 넣어도 됩니다.
const styles = `
  body { transition: background-color .2s ease, color .2s ease; }
  body.dark { background-color: #121212; color: #eaeaea; }
`;

export default function DarkModeToggle() {
  const [dark, setDark] = useState(false);

  useEffect(() => {
    // TODO: dark 값이 바뀔 때마다 body에 dark 클래스를 추가/제거
    // TODO: cleanup에서 항상 body의 dark 클래스를 제거
  }, [dark]);

  return (
    <section style={{ fontFamily: "system-ui", maxWidth: 560, margin: "32px auto" }}>
      <style>{styles}</style>
      <h2>🌙 다크 모드 토글</h2>

      <label style={{
        display: "inline-flex", alignItems: "center", gap: 8,
        padding: "8px 12px", borderRadius: 8, border: "1px solid #ddd"
      }}>
        <input
          type="checkbox"
          checked={dark}
          onChange={(e) => setDark(e.target.checked)}
        />
        다크 모드 켜기
      </label>

      <p style={{ marginTop: 12 }}>
        현재 모드: <strong>{dark ? "Dark" : "Light"}</strong>
      </p>
    </section>
  );
}
```

## 📌 미션의 의의

- 의존성 배열: `[dark]`가 바뀔 때만 effect가 실행되어 불필요한 작업을 줄입니다.
- DOM 적용 시점: effect는 DOM 반영 직후 실행되므로 클래스 토글이 안전합니다.
- cleanup 습관화: 언마운트 시 전역 부작용(전역 클래스, 리스너 등)을 원상복구합니다.

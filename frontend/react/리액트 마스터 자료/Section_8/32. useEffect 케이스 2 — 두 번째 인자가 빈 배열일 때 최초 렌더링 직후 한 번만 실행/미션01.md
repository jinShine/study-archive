# 🟦 처음 한 번만 실행되는 useEffect + Cleanup으로 ESC 닫기 구현

## 📝 설명

이번 미션의 목표는 `useEffect`의 두 번째 인자 `[]` (빈 배열) 동작을 직접 체험하는 것입니다.
React에서 `useEffect(() => { ... }, [])`는 컴포넌트가 처음 화면에 나타난 직후 한 번만 실행되며,
cleanup 함수는 컴포넌트가 사라질 때 실행됩니다.

이를 응용해 ESC 키로 닫히는 모달 창을 만들어보세요.
즉, 컴포넌트가 마운트될 때 ESC 키 이벤트를 등록하고,
언마운트될 때 이벤트를 해제하는 흐름을 완성하는 것이 목표입니다.

## ✅ 요구사항

- `useEffect`의 두 번째 인자로 빈 배열 `[]`을 전달해야 합니다.
- 컴포넌트가 처음 나타날 때 콘솔에 `"ESC 이벤트 등록됨"`이 찍혀야 합니다.
- 컴포넌트가 사라질 때 콘솔에 `"ESC 이벤트 제거됨"`이 찍혀야 합니다.
- `window.addEventListener("keydown", handleEsc)` / `removeEventListener`로 이벤트를 관리해야 합니다.
- ESC 키를 누르면 모달이 닫히도록 (`onClose`) 호출되어야 합니다.

## 🧱 Base Code

아래 코드를 보고 TODO 부분을 채워 완성하세요.

```jsx
import { useEffect } from "react";

export default function Modal({ onClose }) {
  useEffect(() => {
    // TODO 1: handleEsc 함수를 선언하세요.
    // keydown 이벤트가 발생했을 때,
    // e.key가 "Escape"이면 onClose()를 호출해야 합니다.

    // TODO 2: window에 keydown 이벤트를 등록하세요.
    // console.log("ESC 이벤트 등록됨") 도 함께 출력하세요.

    // TODO 3: cleanup 함수에서 이벤트 리스너를 제거하세요.
    // console.log("ESC 이벤트 제거됨") 을 출력하세요.

  }, []); // 빈 배열: 마운트 시 한 번만 실행, 언마운트 시 cleanup 실행

  return (
    <div className="modal">
      <h2>모달 창</h2>
      <p>ESC 키를 누르면 닫힙니다.</p>
    </div>
  );
}
```

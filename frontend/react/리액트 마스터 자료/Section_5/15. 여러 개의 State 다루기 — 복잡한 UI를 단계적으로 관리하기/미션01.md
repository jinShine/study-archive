# 🟥 여러 개의 상태를 독립적으로 관리하기 — 입력·체크박스·셀렉트 동시 제어

## 📝 설명

하나의 컴포넌트 안에서 여러 개의 `state`를 선언하고, 각각을 독립적으로 업데이트하여 화면에 반영해 보세요.

## ✅ 요구사항

- 텍스트 입력창(`text`), 세일 필터 체크박스(`isSale`), 카테고리 셀렉트(`category`)를 모두 controlled component로 만들기
- 현재 선택/입력 상태를 "요약 문구"로 실시간 표시하기
- (선택) "초기화" 버튼으로 세 `state`를 한 번에 기본값으로 되돌리기

## 🖥️ 출력 결과 예시

```text
# 초기 화면
입력한 텍스트: ""
세일만 보기: 꺼짐
카테고리: all

# 텍스트에 "후드티" 입력, 체크박스 체크, 카테고리 electronics 선택 후
입력한 텍스트: "후드티"
세일만 보기: 켜짐
카테고리: electronics
```

## 💡 구현에 필요한 개념

- **`useState`를 여러 번 호출해 서로 다른 타입의 상태(문자열/불리언/선택값)를 선언**

  ```jsx
  const [text, setText] = useState("");
  const [isSale, setIsSale] = useState(false);
  const [category, setCategory] = useState("all");
  ```

- **Controlled input: `value`와 `onChange`로 입력값을 상태에 동기화**

  ```jsx
  <input value={text} onChange={(e) => setText(e.target.value)} />
  ```

- 체크박스는 `e.target.checked`, 셀렉트는 `e.target.value` 사용
- 상태 변경 → 재렌더링 트리거 → React가 Virtual DOM diff 후 변경점만 실제 DOM에 반영
- 상태를 객체 하나로 합치는 대신 독립 `state`로 분리하면 가독성과 유지보수성이 좋아짐

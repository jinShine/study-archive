# 🟨 children과 단방향 데이터 흐름 실습

## 📝 설명

React에서는 모든 데이터가 부모 → 자식 방향으로만 흐릅니다.
부모는 자식에게 props를 전달할 수 있지만, 자식은 부모의 데이터를 직접 바꿀 수 없습니다.

이번 미션에서는 props를 통해 데이터를 전달하면서,
`children`을 이용해 컴포넌트 안에 동적으로 다른 요소를 삽입해보세요.

## 🖥️ 출력 결과 예시

```text
[Profile]
Name: Yongsu
Hello, React Learner!
```

## 💡 구현에 필요한 개념

- **단방향 데이터 흐름 (Unidirectional Data Flow)**
  - React에서는 데이터가 항상 "부모 → 자식"으로만 흐릅니다.
- **`children` props**
  - `<Card> ... </Card>` 형태로 작성하면, 그 사이의 내용이 `props.children`으로 전달됩니다.
- **조합형 컴포넌트 (Composition with children)**
  - 상위 컴포넌트는 하위 컴포넌트에게 구조를 제공하고,
    하위는 실제 내용을 `children`으로 전달받아 렌더링합니다.

# 🟨 나만의 첫 컴포넌트 만들기

## 📝 설명

React에서 함수형 컴포넌트는 UI를 이루는 가장 작은 단위입니다.
이번 미션에서는 여러분만의 `Hello` 컴포넌트를 직접 정의하고, JSX 문법으로 화면에 출력해 봅시다.

## 🖥️ 출력 결과 예시

```text
Hello, React!
```

## 💡 구현에 필요한 개념

- 함수형 컴포넌트의 정의: `function ComponentName() { return <JSX /> }`
- 대문자로 시작하는 컴포넌트 이름 규칙
- JSX 문법과 `ReactDOM.createRoot` / `render` 사용
- 루트 엘리먼트는 반드시 하나여야 함 (`<></>`로 묶을 수도 있음)

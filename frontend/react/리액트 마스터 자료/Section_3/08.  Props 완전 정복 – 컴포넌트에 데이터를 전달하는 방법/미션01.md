# 🟨 Props로 인사 메시지 바꾸기

## 📝 설명

이번에는 같은 컴포넌트지만 입력값(Props) 에 따라 다른 결과가 나오도록 만들어 봅시다.
`Greeting` 컴포넌트가 `name`이라는 props를 받아서 화면에 출력하도록 하세요.

## 🖥️ 출력 결과 예시

```text
Hello, Yongsu!
```

## 💡 구현에 필요한 개념

- Props는 컴포넌트의 첫 번째 인자로 전달되는 객체입니다.
- `<Greeting name="Yongsu" />` → 내부적으로 `Greeting({ name: "Yongsu" })` 호출
- JSX 속성 이름은 camelCase (예: `className`, `onClick`)
- 구조 분해 할당으로 props 사용하기: `function Greeting({ name }) { ... }`
- props는 읽기 전용으로 직접 수정하면 안 됨.

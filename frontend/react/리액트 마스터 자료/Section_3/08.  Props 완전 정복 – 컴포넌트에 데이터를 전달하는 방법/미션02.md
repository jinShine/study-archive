# 🟨 Props 구조 분해와 컴포넌트 합성 실습

## 📝 설명

React의 진짜 강점은 작은 컴포넌트들을 조합(Composition) 하여 큰 UI를 만드는 데 있습니다.
이번 미션에서는 여러 개의 컴포넌트를 만들어 상위 → 하위로 props를 전달하고,
구조 분해(destructuring)로 props를 간결하게 사용하는 연습을 합니다.

## 🖥️ 출력 결과 예시

```text
🧑‍💻 Welcome, Yongsu!
Age: 20
Country: Korea
```

## 💡 구현에 필요한 개념

- **컴포넌트 합성 (Composition)**
  - 하나의 컴포넌트 안에서 다른 컴포넌트를 포함해 사용할 수 있음.
- **Props 구조 분해 할당 (Destructuring)**
  - `function UserInfo({ name, age }) { ... }`
  - 필요한 값만 직접 꺼내어 코드 가독성을 높임.
- **읽기 전용 원칙**
  - props는 부모가 준 데이터를 "참조"만 해야 하며, 수정은 절대 금지.
- **하위 컴포넌트 재사용**
  - `Emoji`, `Title`, `UserInfo`를 조합해서 하나의 `ProfileHeader`를 구성.

# 🟧 React 이벤트 처리 실습 — 클릭된 리스트 항목의 이름과 인덱스 출력하기

## 📝 설명

이번 미션에서는 React에서의 이벤트 처리(event handling) 개념을 직접 실습합니다. React에서는 HTML과 달리, 이벤트를 문자열로 작성하지 않고 camelCase 형태의 속성(`onClick`, `onChange`) 과 함수 참조를 사용해야 합니다.

예를 들어,

```jsx
<!-- HTML 방식 -->
<button onclick="alert('hi')">클릭</button>
```

이 방식 대신, React에서는 다음처럼 작성합니다.

```jsx
<!-- React 방식 -->
<button onClick={handleClick}>클릭</button>
```

이처럼 괄호 없이 함수 참조를 전달해야 클릭 시점에 실행됩니다. 괄호를 써버리면 렌더링할 때 즉시 실행되어 의도한 대로 동작하지 않습니다.

또한 React의 이벤트 핸들러는 브라우저의 원래 이벤트가 아니라 `SyntheticEvent`(합성 이벤트) 를 받습니다.
`SyntheticEvent`는 모든 브라우저에서 동일하게 작동하도록 React가 래핑한 이벤트 객체로, `event.target.value`나 `event.clientX`처럼 일관된 속성을 제공합니다.

이번 미션에서는 리스트 항목을 클릭할 때, 클릭된 과일 이름과 인덱스를 콘솔에 출력하는 기능을 구현해 보세요.

## 🖥️ 출력 결과 예시

```text
# 리스트 항목 "바나나" 클릭 시 콘솔 출력
클릭된 항목: 바나나
인덱스: 1
SyntheticEvent 객체: SyntheticBaseEvent {...}
```

## ✅ 요구사항

- `onClick` 속성을 camelCase로 작성하기
- JSX 안에서는 함수 참조만 전달 (`onClick={handleClick}`)
- `map()` 내부에서 항목 이름(`item`)과 인덱스(`index`)를 함께 전달하기
- `SyntheticEvent` 객체를 활용해 이벤트 정보를 일관성 있게 다루기

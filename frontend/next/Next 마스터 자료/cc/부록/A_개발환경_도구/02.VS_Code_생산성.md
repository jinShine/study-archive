## 1. 생존 단축키 TOP 20 🔴

> 💡 이 20개만 외우면 코딩 속도가 **2배** 빨라진다!
> Mac은 `Cmd`, Windows/Linux는 `Ctrl`로 읽으면 돼.

### 파일 & 탐색 (가장 많이 사용!)

```
Cmd + P           파일 빠른 열기 ⭐⭐⭐ (가장 중요!)
                  → 파일명 몇 글자만 치면 즉시 이동!
                  → 마우스로 파일 탐색기를 클릭하는 건 시간 낭비!

Cmd + Shift + P   명령 팔레트 ⭐⭐⭐
                  → VS Code의 모든 기능을 검색으로!
                  → "format", "terminal", "theme" 등 뭐든!

Cmd + B           사이드바 토글 (접기/펼치기)
                  → 코딩할 때 사이드바 접으면 화면이 넓어짐!

Cmd + `           터미널 토글 ⭐⭐
                  → 터미널을 빠르게 열고 닫기!

Cmd + \           에디터 분할 (좌우로 나누기!)
                  → 코드 2개를 동시에 보면서 작업!
```

### 편집 (코드 작성 속도 UP!)

```
Cmd + D           같은 단어 하나씩 선택 ⭐⭐⭐
                  → 변수명 일괄 수정할 때! "count"를 전부 "total"로!

Cmd + Shift + L   같은 단어 전부 선택 ⭐⭐
                  → Cmd+D를 여러 번 누르는 것의 한 방 버전!

Alt + ↑/↓         줄 이동 ⭐⭐
                  → 현재 줄을 위/아래로 이동! 잘라내기+붙여넣기 필요 없음!

Cmd + Shift + K   줄 삭제 ⭐⭐
                  → 현재 줄을 통째로 삭제! (전체 선택 후 Delete 필요 없음!)

Alt + Shift + ↓   줄 복사 ⭐
                  → 현재 줄을 아래로 복사! 비슷한 코드 반복할 때!

Cmd + /           주석 토글 ⭐⭐
                  → 선택한 줄을 주석 처리/해제!

Cmd + Shift + F   전체 프로젝트에서 검색 ⭐⭐
                  → "이 함수를 어디서 쓰고 있지?" 찾기!

F2                이름 변경 (Rename Symbol) ⭐⭐
                  → 변수/함수 이름을 바꾸면 사용하는 곳 전부 자동 변경!
                  → 일일이 Cmd+D로 찾을 필요 없음!
```

### 코드 정리

```
Cmd + Shift + I   코드 포맷팅 (Prettier 연동!) ⭐⭐
                  → 들여쓰기, 줄바꿈이 자동으로 예쁘게!

Cmd + .           Quick Fix (빠른 수정) ⭐⭐
                  → 에러에 커서를 놓고 누르면 자동 수정 제안!
                  → import 자동 추가, 타입 추론 등!

Cmd + K, Cmd + F  선택 영역만 포맷팅

Cmd + Z           실행 취소 (당연!)
Cmd + Shift + Z   다시 실행 (Redo)
```

### 네비게이션

```
Cmd + Click       정의로 이동 ⭐⭐⭐
                  → 함수/변수를 Cmd+Click하면 정의된 파일로 점프!
                  → "이 함수가 어떻게 구현되어 있지?" 바로 확인!

Cmd + ← / →       줄의 처음/끝으로 이동
Alt + ← / →       단어 단위로 이동
```

## 2. 멀티 커서 — 마법의 동시 편집! 🟡

> 💡 커서가 여러 개! 한 번에 여러 줄을 동시에 타이핑!

```
Alt + Click        원하는 위치에 커서 추가!
                   → 3곳을 클릭하면 3개의 커서!
                   → 타이핑하면 3곳에 동시에 입력!

Cmd + Alt + ↑/↓   위/아래에 커서 추가!
                   → 연속된 줄에 동시에 작업!

예시:
  const name = "";     ← 커서 1
  const email = "";    ← 커서 2
  const phone = "";    ← 커서 3
  → 3개에 동시에 타이핑 가능!
```

## 3. 필수 확장 프로그램 TOP 10 🔴

```
⭐⭐⭐ 반드시 설치:
  1. ESLint           → 코드 문법 오류 실시간 감지!
  2. Prettier         → 코드 자동 포맷팅! (저장 시 자동!)
  3. Tailwind CSS IntelliSense → 클래스명 자동완성! ⭐
  4. ES7+ Snippets    → 컴포넌트 스니펫! (rafce 치면 자동 생성!)

⭐⭐ 강력 추천:
  5. Auto Rename Tag  → <div>를 <section>으로 바꾸면 </div>도 자동!
  6. Error Lens       → 에러를 줄 옆에 바로 표시! (하단 패널 안 봐도 됨!)
  7. GitLens          → 각 줄이 누가 언제 수정했는지! (blame!)
  8. Pretty TypeScript Errors → TS 에러를 읽기 쉽게 번역!

⭐ 유용:
  9. Material Icon Theme  → 파일 아이콘이 예뻐짐! (직관적!)
  10. Korean Language Pack → VS Code 한국어화!
```

### Tailwind CSS IntelliSense 활용법

```
이 확장이 없으면:
  className="bg-" → 뭐가 있는지 모름! 문서 찾아야 함!

이 확장이 있으면:
  className="bg-" → bg-red-500, bg-blue-300... 자동완성 목록!
  → 색상 미리보기까지 옆에 표시! 🎨
  → 호버하면 실제 CSS 속성이 뭔지 보여줌!
```

### ES7+ Snippets 자주 쓰는 것

```
에디터에 약자를 치고 Tab:

  rafce  → React Arrow Function Component (export default)
  useState → useState 스니펫
  useEffect → useEffect 스니펫
  tsrafce → TypeScript React Arrow Function Component

rafce + Tab 결과:
  const Component = () => {
    return (
      <div>Component</div>
    )
  }
  export default Component
```

## 4. settings.json — 핵심 설정 🔴

> `Cmd + Shift + P` → "Settings JSON" 검색 → "Open User Settings (JSON)"

```json
{
  // 저장할 때 자동으로 Prettier 포맷팅!
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",

  // 저장할 때 자동으로 ESLint 수정!
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": "explicit"
  },

  // 폰트 크기 (취향!)
  "editor.fontSize": 14,
  "editor.lineHeight": 1.7,

  // 탭 크기 = 2칸 (프론트엔드 표준!)
  "editor.tabSize": 2,

  // 미니맵 끄기 (화면 넓게!)
  "editor.minimap.enabled": false,

  // 빵 부스러기(breadcrumb) 끄기
  "breadcrumbs.enabled": false,

  // 파일 저장 시 마지막 줄에 빈 줄 추가 (Git 컨벤션!)
  "files.insertFinalNewline": true,

  // Tailwind CSS 정렬 (클래스 순서 자동 정리!)
  "tailwindCSS.experimental.classRegex": [
    ["cva\\(([^)]*)\\)", "[\"'`]([^\"'`]*).*?[\"'`]"]
  ],

  // TypeScript import 자동 업데이트
  "typescript.updateImportsOnFileMove.enabled": "always",

  // 터미널 기본 쉘 (Mac)
  "terminal.integrated.defaultProfile.osx": "zsh",

  // Emmet (HTML 약어 확장) → JSX에서도!
  "emmet.includeLanguages": {
    "typescriptreact": "html"
  }
}
```

## 5. 터미널 활용 🔴

```
VS Code 내장 터미널 = 별도 터미널 앱 불필요!

Cmd + `                터미널 열기/닫기!
Cmd + Shift + `        새 터미널 탭 추가!
Cmd + \  (터미널에서)   터미널 분할! (2개 동시에!)

실무 활용:
  터미널 1: npm run dev (Next.js 서버!)
  터미널 2: npx playwright test --ui (테스트!)
  터미널 3: git 명령어!

→ 3개를 동시에 돌리면서 코딩!
```

## 6. 디버깅 기초 🟡

```
F5              디버깅 시작!
F9              브레이크포인트 설정 (이 줄에서 멈춰!)
F10             다음 줄로 이동 (Step Over)
F11             함수 안으로 들어가기 (Step Into)
Shift + F11     함수 밖으로 나오기 (Step Out)

브레이크포인트:
  줄 번호 왼쪽 클릭 → 빨간 점!
  → 코드가 이 줄에서 멈춤!
  → 변수 값을 확인할 수 있음!

Next.js 디버깅:
  .vscode/launch.json 파일에 서버 사이드 디버깅 설정 가능!
  하지만 대부분 console.log + 브라우저 DevTools로 충분! 😄
```

## 7. 설정 동기화 🟡

```
GitHub 계정으로 VS Code 설정 동기화!

  1. 좌하단 사람 아이콘 → "Turn on Settings Sync"
  2. GitHub으로 로그인
  3. 동기화할 항목 선택 (전부 체크!)
  4. 완료!

이점:
  → 새 컴퓨터에서 VS Code 깔고 로그인만 하면!
  → 확장, 설정, 단축키, 스니펫이 전부 복원! ⭐
  → "다시 세팅하느라 반나절" 안 해도 됨!
```

# 10. Flexbox 완전 정복

> **키워드**: `display: flex` `주축/교차축` `justify-content` `align-items` `flex-grow` `gap`

| 등급 | 섹션 | 의미 |
|------|------|------|
| 🔴 필수 | 축 개념, 정렬 속성, flex shorthand, gap | 면접 단골 + 실무 90% |
| 🟡 이해 | flex-wrap, align-self, order, 실전 패턴 | 복잡한 레이아웃 |
| 🟢 참고 | flex-basis 계산, min-width: 0 트릭 | 디버깅 고급 |

---

## 1. Flexbox란? 🔴

> 💡 **비유: 빨랫줄에 빨래 널기**
> `display: flex`를 선언하면 부모가 **빨랫줄**이 되고, 자식 요소들이 **빨래**가 된다.
> 빨래들은 줄을 따라 한 방향으로 나란히 배치된다.

```css
.container {
  display: flex;  /* 이 한 줄이면 자식들이 가로로 나란히! */
}
```

```
display: flex 적용 전 (블록 요소 = 세로 쌓임)
┌───────────────────┐
│      Box 1        │
├───────────────────┤
│      Box 2        │
├───────────────────┤
│      Box 3        │
└───────────────────┘

display: flex 적용 후 (가로 나란히!)
┌──────┐ ┌──────┐ ┌──────┐
│ Box1 │ │ Box2 │ │ Box3 │
└──────┘ └──────┘ └──────┘
```

## 2. 핵심 개념: 주축(Main Axis)과 교차축(Cross Axis) 🔴

> ⚠️ 이것을 이해하지 못하면 Flexbox를 절대 마스터할 수 없다.

```
flex-direction: row (기본값)

         주축 (Main Axis) →→→→→→→→→→
         ┌──────┐ ┌──────┐ ┌──────┐
교차축 ↓  │ Box1 │ │ Box2 │ │ Box3 │
(Cross)↓  └──────┘ └──────┘ └──────┘


flex-direction: column

교차축 →  ┌──────────────┐
(Cross)→  │    Box 1     │
          ├──────────────┤  주축
          │    Box 2     │  (Main Axis)
          ├──────────────┤  ↓↓↓
          │    Box 3     │
          └──────────────┘
```

```css
.container {
  display: flex;
  flex-direction: row;       /* 기본값: 가로 배치 (→) */
  flex-direction: row-reverse;   /* 가로 역순 (←) */
  flex-direction: column;    /* 세로 배치 (↓) */
  flex-direction: column-reverse; /* 세로 역순 (↑) */
}
```

> 💡 **핵심 규칙**:
> - `justify-content` → **주축** 방향 정렬
> - `align-items` → **교차축** 방향 정렬
> - `flex-direction`이 바뀌면 축도 바뀐다!

## 3. 부모(컨테이너) 속성 🔴

### justify-content — 주축 정렬

```
justify-content: flex-start (기본값)
|Box1|Box2|Box3|                    |

justify-content: center
|           |Box1|Box2|Box3|           |

justify-content: flex-end
|                    |Box1|Box2|Box3|

justify-content: space-between
|Box1|         |Box2|         |Box3|

justify-content: space-around
|  |Box1|    |Box2|    |Box3|  |

justify-content: space-evenly
|   |Box1|   |Box2|   |Box3|   |
```

### align-items — 교차축 정렬

```css
.container {
  display: flex;
  height: 300px;

  align-items: stretch;     /* 기본값: 자식이 교차축 방향으로 쭉 늘어남 */
  align-items: flex-start;  /* 교차축 시작점 (위쪽) */
  align-items: center;      /* 교차축 중앙 (수직 중앙!) */
  align-items: flex-end;    /* 교차축 끝점 (아래쪽) */
  align-items: baseline;    /* 텍스트 기준선 맞춤 */
}
```

### 완벽한 중앙 정렬 (면접 단골!)

```css
/* Flexbox로 가로+세로 완벽 중앙 정렬 */
.center {
  display: flex;
  justify-content: center;  /* 주축(가로) 중앙 */
  align-items: center;      /* 교차축(세로) 중앙 */
  height: 100vh;            /* 화면 전체 높이 */
}
```

### gap — 자식 간 간격 🔴

```css
.container {
  display: flex;
  gap: 16px;            /* 모든 방향 16px */
  gap: 20px 10px;       /* row-gap 20px, column-gap 10px */
}
```

> 💡 **gap은 margin 대체제**다. 자식에 margin을 줄 필요 없이 부모에 gap만 주면 된다.
> 마지막 요소 뒤에 불필요한 margin이 생기는 문제도 없다!

### flex-wrap — 줄바꿈 🟡

```css
.container {
  display: flex;
  flex-wrap: nowrap;   /* 기본값: 한 줄에 억지로 꿰맞춤 (넘침 발생) */
  flex-wrap: wrap;     /* 넘치면 다음 줄로 (반응형 그리드 대용!) */
}
```

```
flex-wrap: wrap
┌──────┐ ┌──────┐ ┌──────┐
│ Box1 │ │ Box2 │ │ Box3 │   ← 한 줄에 3개
└──────┘ └──────┘ └──────┘
┌──────┐ ┌──────┐
│ Box4 │ │ Box5 │             ← 넘치면 다음 줄로
└──────┘ └──────┘
```

## 4. 자식(아이템) 속성 🔴

### flex-grow / flex-shrink / flex-basis

```css
.item {
  flex-grow: 0;    /* 기본값: 남는 공간을 차지하지 않음 */
  flex-grow: 1;    /* 남는 공간을 균등하게 차지 */
  flex-grow: 2;    /* 다른 1짜리보다 2배 더 차지 */

  flex-shrink: 1;  /* 기본값: 공간 부족 시 줄어듦 */
  flex-shrink: 0;  /* 절대 줄어들지 않음 (고정 너비 유지) */

  flex-basis: auto; /* 기본값: width 또는 콘텐츠 크기 */
  flex-basis: 200px; /* 시작 크기 200px (width 대신 사용) */
}

/* ✅ 단축 속성 (실무에서 가장 많이 사용) */
.item { flex: 1; }           /* flex-grow:1, shrink:1, basis:0% → 균등 분배 */
.item { flex: 0 0 200px; }   /* grow:0, shrink:0, basis:200px → 고정 200px */
.item { flex: none; }        /* grow:0, shrink:0, basis:auto → 콘텐츠 크기 고정 */
```

```
flex-grow 비율 분배 예시:

컨테이너 너비: 600px
┌─────────────┬─────────────────────────────┐
│  A (flex:1)  │        B (flex:2)            │
│   200px      │         400px                │
└─────────────┴─────────────────────────────┘
→ 1:2 비율로 공간 분배
```

### align-self — 개별 아이템 교차축 정렬

```css
.special {
  align-self: center;  /* 이 아이템만 교차축 중앙 */
  /* 부모의 align-items를 이 아이템만 무시하고 독자적으로 설정 */
}
```

### order — 시각적 순서 변경

```css
.first { order: 3; }   /* HTML 순서는 1번이지만 화면에서 3번째 */
.last { order: -1; }   /* 가장 앞으로 이동 (기본값은 0) */
```

> ⚠️ `order`는 시각적 순서만 바꾸고, 스크린 리더의 읽기 순서(DOM 순서)는 바뀌지 않는다.

## 5. 실전 레이아웃 패턴 5가지 🟡

### 패턴 1: 네비게이션 바 (로고 왼쪽, 메뉴 오른쪽)

```css
.navbar {
  display: flex;
  justify-content: space-between;  /* 양쪽 끝으로 밀기 */
  align-items: center;
  padding: 0 20px;
  height: 60px;
}
```

### 패턴 2: 카드 그리드 (반응형)

```css
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}
.card {
  flex: 1 1 300px;  /* 최소 300px, 남는 공간 균등 분배 */
  /* 화면이 좁으면 한 줄에 1개, 넓으면 3~4개 */
}
```

### 패턴 3: 사이드바 + 메인 콘텐츠

```css
.layout {
  display: flex;
  min-height: 100vh;
}
.sidebar {
  flex: 0 0 250px;  /* 고정 250px (안 줄어듦) */
}
.main {
  flex: 1;           /* 나머지 전체 차지 */
}
```

### 패턴 4: 푸터를 페이지 바닥에 고정 (Sticky Footer)

```css
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}
main {
  flex: 1;   /* main이 남은 공간을 모두 차지 → footer가 바닥으로 밀림 */
}
footer {
  /* flex-shrink: 0이 기본이므로 줄어들지 않음 */
}
```

### 패턴 5: 입력 필드 + 버튼

```css
.search-bar {
  display: flex;
  gap: 8px;
}
.search-bar input {
  flex: 1;        /* 남는 공간 전체 차지 */
}
.search-bar button {
  flex: none;     /* 버튼은 콘텐츠 크기 고정 */
}
```

## 6. 실습: Flexbox로 완성하는 3가지 레이아웃

```
과제 1: 네비게이션 바
  - 로고(왼쪽) + 메뉴 3개(오른쪽) + 수직 중앙 정렬
  - gap으로 메뉴 간격

과제 2: 카드 그리드
  - 카드 6개를 flex-wrap으로 반응형 배치
  - 각 카드 최소 250px, 남는 공간 균등 분배
  - 카드 내부도 flex로 구성 (이미지 + 텍스트 + 버튼)

과제 3: Holy Grail 레이아웃
  - 헤더(상단 고정)
  - 사이드바(왼쪽 200px 고정) + 메인(가변) + 사이드바(오른쪽 200px 고정)
  - 푸터(바닥 고정)
  - min-height: 100vh
```

## 면접 대비

### 🔴 필수 질문

**Q1. Flexbox의 주축과 교차축이란?**

> 주축(Main Axis)은 `flex-direction`이 결정하는 아이템 배치 방향이다. 기본값 `row`에서는 가로가 주축이다. 교차축(Cross Axis)은 주축에 수직인 방향이다. `justify-content`는 주축, `align-items`는 교차축 방향의 정렬을 제어한다. `flex-direction: column`으로 바꾸면 두 축이 뒤바뀐다.

**Q2. `flex: 1`은 정확히 무엇을 의미하나요?**

> `flex: 1`은 `flex-grow: 1`, `flex-shrink: 1`, `flex-basis: 0%`의 단축 속성이다. 아이템이 사용 가능한 남은 공간을 균등하게 나눠 가지며, 공간이 부족하면 줄어들 수도 있다. `flex-basis: 0%`이므로 콘텐츠 크기와 무관하게 동일한 비율로 공간을 분배한다.

**Q3. CSS로 요소를 완벽하게 가로+세로 중앙 정렬하는 방법은?**

> `display: flex; justify-content: center; align-items: center;`를 부모에 선언한다. 부모에 높이(`height` 또는 `min-height`)가 설정되어 있어야 세로 중앙이 작동한다. 또는 `display: grid; place-items: center;`로도 한 줄로 가능하다.

## 핵심 요약

```
┌────────────────────────────────────────────────┐
│           Flexbox 핵심 요약                     │
├────────────────────────────────────────────────┤
│                                                │
│  부모: display: flex;                           │
│    flex-direction: row(가로) / column(세로)      │
│    justify-content: 주축 정렬                   │
│    align-items: 교차축 정렬                     │
│    gap: 자식 간 간격                            │
│    flex-wrap: wrap (줄바꿈)                     │
│                                                │
│  자식:                                          │
│    flex: 1       → 남는 공간 균등 분배           │
│    flex: 0 0 Npx → 고정 크기                    │
│    flex: none    → 콘텐츠 크기 유지              │
│    align-self    → 개별 교차축 정렬              │
│                                                │
│  완벽 중앙: justify-content + align-items center │
│                                                │
└────────────────────────────────────────────────┘
```

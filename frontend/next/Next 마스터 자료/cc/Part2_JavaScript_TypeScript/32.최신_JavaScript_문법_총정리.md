```js

// 2. 구조 분해 할당
const { name, age, ...rest } = user;
```

## 2. ES2020+ 실무에서 쓰는 최신 문법 🟡

### structuredClone — 깊은 복사 표준 (2022)

```js
const original = { a: 1, b: { c: 2 }, d: [3, 4] };
const deep = structuredClone(original);
deep.b.c = 99;
console.log(original.b.c);  // 2 ✅ (완전 독립!)
```

### Array.at() — 음수 인덱스 (2022)

```js
const arr = [10, 20, 30, 40, 50];
arr.at(0);    // 10
arr.at(-1);   // 50 ← 마지막 요소!
arr.at(-2);   // 40
// arr[-1]은 undefined! at()을 써야 함
```

### 비파괴 배열 메서드 (2023) — React 친화적!

```js
const arr = [3, 1, 2];

// 원본 변경 메서드 → 비파괴 버전
arr.sort()      → arr.toSorted()       // 새 배열 반환 ✅
arr.reverse()   → arr.toReversed()     // 새 배열 반환 ✅
arr.splice(1,1) → arr.toSpliced(1, 1)  // 새 배열 반환 ✅

// with() — 특정 인덱스만 교체 (새 배열!)
const newArr = arr.with(1, 99);  // [3, 99, 2] (index 1을 99로)
```

> 💡 React 상태 업데이트에서 `[...arr].sort()` 대신 `arr.toSorted()` 가능!

### Object.groupBy() (2024)

```js
const people = [
  { name: "철수", age: 25 },
  { name: "영희", age: 30 },
  { name: "민수", age: 25 },
];

const grouped = Object.groupBy(people, person =>
  person.age >= 30 ? "senior" : "junior"
);
// { junior: [{name:"철수",...}, {name:"민수",...}], senior: [{name:"영희",...}] }
```

### Top-level await (2022)

```js
// 이전: async 함수 안에서만 await 사용 가능
// 현재: 모듈 최상위에서 바로 await 가능!
const data = await fetch("/api/config").then(r => r.json());
export default data;
```

## 3. 알아두면 좋은 것들 🟢

```js
// 논리 할당 연산자 (2021)
x ??= 10;    // x가 null/undefined면 10 할당 (x = x ?? 10)
x ||= 10;    // x가 falsy면 10 할당
x &&= fn(x); // x가 truthy면 fn(x) 결과 할당

// Promise.any (2021) — 하나라도 성공하면 성공
await Promise.any([fetch(cdn1), fetch(cdn2), fetch(cdn3)]);

// String.replaceAll (2021)
"a-b-c".replaceAll("-", "/");  // "a/b/c"

// Error.cause (2022) — 에러 체이닝
throw new Error("DB 저장 실패", { cause: originalError });

// Hashbang (#!) (2023) — Node.js 스크립트 실행
#!/usr/bin/env node
```

### 🟡 개념 질문

**Q1. ES6에서 가장 중요한 문법 변경 3가지?**

> 첫째, `let`/`const` — 블록 스코프 변수 선언으로 `var`의 문제를 해결. 둘째, 화살표 함수 — 간결한 문법과 렉시컬 `this` 바인딩. 셋째, 모듈 시스템(`import`/`export`) — 코드 분리와 트리 셰이킹을 가능하게 함. 추가로 구조 분해, 스프레드, 템플릿 리터럴, Promise도 핵심이다.

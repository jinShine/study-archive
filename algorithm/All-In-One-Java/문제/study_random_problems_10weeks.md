**문제**: x가 자신의 각 자릿수의 합으로 나누어떨어지면 true를 반환하세요.

```
x = 18 → true (1+8=9, 18%9=0)
x = 19 → false
```

**핵심**: 자릿수 추출, 나머지 연산

### Problem 2: 순위
**링크**: https://school.programmers.co.kr/learn/courses/30/lessons/49191  
**난이도**: 🟡 **Lv.2**  
**주제**: 그래프, DFS, 경로 탐색  

**문제**: 경기 결과가 주어질 때, 정확하게 순위를 매길 수 있는 선수의 수를 구하세요.

```
n = 5, results = [[4,3],[4,2],[3,2],[1,2],[2,5]]
→ 3 (1,2,5번 선수가 순위 확정)
```

**핵심**:
- 그래프로 경기 결과 표현
- DFS로 도달 가능한 노드 세기

### Problem 3: 입국심사
**링크**: https://school.programmers.co.kr/learn/courses/30/lessons/43238  
**난이도**: 🔴 **Lv.3**  
**주제**: 이진탐색, 그리디  

**문제**: 심사관별 심사 시간이 주어질 때, n명을 모두 심사하는 최소 시간을 구하세요.

```
n = 6, times = [7, 10]
→ 28 (7분 심사관 4명, 10분 심사관 2명 이용)
```

**핵심**:
- 이진탐색으로 최소 시간 찾기
- 주어진 시간에 몇 명을 심사할 수 있는지 계산

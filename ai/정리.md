# 클로드 코드로 시작하는 실전 에이전틱 코딩 (완벽한 통제를 위한 AI 개발팀 구축 가이드)

에이전틱 코딩(Agentic Coding)
- AI가 파일 시스템 탐색, 코드 수정, 테스트 실행까지 자율로 수행하는 접근 방식입니다.
- 앤트로픽은 이 과정일 `탐색-계획-구현-커밋 워크플로`로 정의한다.

에이전틱 루프(Agentic Loop)
- 클로드 코드는 에이전틱 루프라는 순환 구조로 작동한다.
- 사용자의 질문에 한 번 응답하고 끝나는 일회성 대화와 달리, 컨텍스트를 수집하고, 작업을 수행하고, 결과를 확인한 뒤 문제가 있으면 다시 처음부터 돌아간다.
- 단계:
    - 1단계: 컨텍스트 수집
    - 2단계: 작업수행
    - 3단계: 결과 확인
